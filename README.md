# Stock_price_prediction_ML-
Stock price prediction using LSTM on historical data

This project predicts stock prices using a Long Short-Term Memory (LSTM) model based on historical stock data.

Features
- Uses historical stock data from yfinance
- Multi-feature input (Open, High, Low, Close, Volume)
- 60-day time window for prediction
- LSTM-based model
- Actual vs predicted price visualization
- Next-day stock price prediction

Tech Stack
- Python
- TensorFlow / Keras
- Pandas, NumPy
- Matplotlib
- yfinance

How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells
